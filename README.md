# Jogo-da-velha
Projeto de Estrutura de Dados (Unicesumar): Jogo da Velha em C. Aplica structs e ponteiros para persistência de dados. Implementa login mascarado, ranking de estatísticas e modos de jogo (PVP, Bot, Sem Login). Permite resgatar partidas (partidas.txt). Interface com cores (windows.h).
